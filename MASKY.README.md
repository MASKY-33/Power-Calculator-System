## Power Calculator System
Dit systeem vraagt de gebruiker om een getal en berekent vervolgens 10.0 ** getal.
Het gebruikt try/except om zowel foutieve input als extreem grote waarden veilig af te handelen.

## Features
- Draait continu tot de gebruiker "stop" typt
- Zet invoer om naar een integer
- Berekent machtsverheffing: 10.0 ** digit
- Foutafhandeling voor:
- ValueError → geen geldig nummer
- OverflowError → getal te groot voor berekening

# How to use
Typ een getal om de macht van 10 te berekenen.
Foutieve input of te grote waarden geven een duidelijke melding.
Typ "stop" om te stoppen.
