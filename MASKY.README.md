# Power Calculator System
This system asks the user for a number and calculates 10.0 ** number.
It uses exception handling to deal with invalid input and extremely large values.

## Features
- Runs until the user types "stop"
- Converts input to an integer
- Calculates powers of 10
- Handles invalid input (ValueError)
- Handles overly large numbers (OverflowError)

# How to use
Enter any number to calculate 10.0 ** number.
Invalid or huge inputs trigger clear error messages.
Type "stop" to exit.


## Learning Purpose
Practice with:
- exception handling
- power calculations
- input validation
- loops
